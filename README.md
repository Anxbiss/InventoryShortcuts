# InventoryShortcuts

InventoryShortcuts fügt deinem Survival-Inventar praktische Shortcut-Tabs hinzu, mit denen du häufig genutzte Commands direkt per Klick ausführen kannst.

Die Shortcuts lassen sich über eine Config anpassen: Du kannst eigene Commands, Labels und Item-Icons für die Buttons festlegen. Zusätzlich zeigt die Mod optional unten rechts ein kleines Mondphasen- und Wetter-Display an.

Features

Anpassbare Shortcut-Buttons im Inventar
Obere und untere Shortcut-Leiste
Eigene Commands pro Button
Eigene Item-Icons als Thumbnail
Config-Menü über Hotkey und optional Mod Menu
Optionales Mondphasen- und Wetter-Display
Clientseitig und für Fabric
Benötigt

Fabric Loader
[Fabric API](https://modrinth.com/mod/fabric-api)
Optional:

[Mod Menu](https://modrinth.com/mod/modmenu) für einfacheren Zugriff auf die Config
